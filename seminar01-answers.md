## Семинар 1
### Задача 1

Дано:

* $n = 40$, $m = 52$, $s_1 = 32$, $s_2 = 40$
* $\alpha = 0.10$, двусторонняя альтернатива, сравниваем доли

Решение:

* Гипотезы: $H_0: p_1 = p_2$ против $H_1: p_1 \ne p_2$
* Выборочные оценки: $\hat{p}_1 = 0.8$, $\hat{p}_2 = 0.77$, $\hat{p} = 0.78$
* Статистический критерий: z-критерий для сравнения двух долей
* Наблюдаемое значение статистики критерия: $z_{набл} = 0.34$
* P-value: $\text{p-value} = 2\text{P}(Z > z_{набл}) = 2 \cdot \text{P}(Z > 0.34) = 0.7338$
* Вывод: $\text{p-value} > \alpha = 0.10$, $H_0$ не отвергается, доли равны
* Если изменим тип альтернативы на левостороннюю, p-value сократится вдвое, но вывод не изменится

**Дополнение 1.** Если строить критическую область, $z_{крит} = \pm z_{1-\alpha/2} = \pm 1.64$, критическая область: $(-\infty, -1.64] \cup [1.64, +\infty)$, 
наблюдаемое значение статистики в нее не попадает, $H_0$ не отвергается, доли равны.

**Дополнение 2.** В Python практически то же самое, чуть поточнее – можете проверить (по умолчанию двусторонняя альтернатива, 
далее сами можем изменить на левостороннюю):

```{python}
from statsmodels.stats.proportion import proportions_ztest

proportions_ztest(count = [32, 40], nobs = [40, 52])
proportions_ztest(count = [32, 40], nobs = [40, 52], alternative = "smaller")
```

### Задача 2

Дано:

* $n = 500$, $m = 200$, $\hat{p}_1 = 0.02$, $\hat{p}_2 = 0.04$
* $\alpha = 0.1$, односторонняя альтернатива, сравниваем доли

Решение:

* Гипотезы: $H_0: p_1 = p_2$ против $H_1: p_1 < p_2$
* Выборочные оценки: $\hat{p}_1 = 0.02$, $\hat{p}_2 = 0.04$, $\hat{p} = 0.026$
* Статистический критерий: z-критерий для сравнения двух долей
* Наблюдаемое значение статистики критерия: $z_{набл} = -1.54$
* P-value: $\text{p-value} = \text{P}(Z < z_{набл}) = \text{P}(Z < -1.54) = 0.0618$
* Вывод: $\text{p-value} < \alpha = 0.10$, $H_0$ отвергается, вторая доля больше, новый сайт лучше

**Дополнение.** Если строить критическую область, $z_{крит} = -z_{1-\alpha} = -1.28$, критическая область: $(-\infty, -1.28]$, 
наблюдаемое значение статистики в нее попадает, $H_0$ отвергается, вторая доля больше.

### Задача 3

Дано:

* $n = 50$, $m = 60$, $\bar{x}_1 = 3.8$, $\bar{x}_2 = 3.6$, $s_1^2 = 5.76$, $s_2^2 = 6.25$
* $\alpha = 0.01$, односторонняя альтернатива, сравниваем средние (в предположении нормальности распределений и равенства дисперсий)

Решение:

* Гипотезы: $H_0: a_1 = a_2$ против $H_1: a_1 > a_2$
* Выборочные оценки: $\bar{x}_1 = 3.8$, $\bar{x}_2 = 3.6$, $s_1^2 = 5.76$, $s_2^2 = 6.25$, $s = 2.46$
* Статистический критерий: двухвыборочный t-критерий Стьюдента для независимых выборок
* Наблюдаемое значение статистики критерия: $t_{набл} = 0.42$
* P-value: $\text{p-value} = \text{P}(T > t_{набл}) = \text{P}(T > 0.42) \approx 0.3372$
* Вывод: $\text{p-value} > \alpha = 0.01$, $H_0$ не отвергается, средние равны, вторая диета не лучше

**Дополнение.** В Python практически то же самое, чуть поточнее – можете проверить (по умолчанию двусторонняя альтернатива, 
можем изменить на правостороннюю):

```{python}
from scipy import stats

stats.ttest_ind_from_stats(mean1 = 3.8, std1 = 5.76 ** 0.5, 
                           mean2 = 3.6, std2 = 6.25 ** 0.5,
                           nobs1 = 50, nobs2 = 60, 
                           equal_var = True, alternative = "greater")
```

### Задача 4

Дано:

* $n = 6$, $m = 5$, $\bar{x}_1 = 5$, $\bar{x}_2 = 3$, $s_1^2 = 4$, $s_2^2 = 3$
* $\alpha = 0.05$, двусторонняя альтернатива, сравниваем средние (в предположении нормальности распределений и равенства дисперсий)

Решение:

* Гипотезы: $H_0: a_1 = a_2 \text{ }(\sigma^2_1 = \sigma^2_2)$ против $H_1: a_1 \ne a_2 \text{ }(\sigma^2_1 = \sigma^2_2)$
* Выборочные оценки: $\bar{x}_1 = 5$, $\bar{x}_2 = 3$, $s_1^2 = 4$, $s_2^2 = 3$, $s = 1.89$
* Статистический критерий: двухвыборочный t-критерий Стьюдента для независимых выборок
* Наблюдаемое значение статистики критерия: $t_{набл} = 1.75$
* Критическое значение статистики критерия: $t_{крит} = \pm t_{\text{df = n+m-2}, \alpha/2} =
  t_{крит} = \pm t_{\text{df = 9}, 0.025} = \pm 2.2622$, критическая область $(-\infty, -2.26] \cup [2.26, +\infty)$, 
наблюдаемое значение статистики в нее не попадает, $H_0$ не отвергается, средние равны

Точное $\text{pvalue} = 2\text{P}(T > t_{набл})$ можем оценить снизу – по таблице стандартного нормального 
распределения найдем 2P(Z > 1.75) \approx 0.08 и заключим, что 2P(T > 1.75) еще больше.

### Задача 5

Дано:

* $n = 16$, $m = 15$, $s_1 = 10$, $s_2 = 5$
* односторонняя альтернатива, сравниваем дисперсии

Решение:

* Гипотезы: $H_0: \sigma^2_1 = \sigma^2_2$ против $H_1: \sigma^2_1 > \sigma^2_2$
* Статистический критерий: F-критерий Фишера для сравнения двух дисперсий
* Наблюдаемое значение статистики критерия: $F_{набл} = 100/25 = 4$

a) $F_{крит} = F(\text{df}_1 = 15, \text{df}_2 = 14, \alpha = 0.1) = 2.01$, критическая область $[2.01; +\infty)$, наблюдаемое значение лежит в ней, $H_0$ отвергается, первая дисперсия больше второй

b) $F_{крит} = F(\text{df}_1 = 15, \text{df}_2 = 14, \alpha = 0.05) = 2.463$, критическая область $[2.463; +\infty)$, наблюдаемое значение лежит в ней, $H_0$ отвергается, первая дисперсия больше второй

**Дополнение.** Можно ли использовать здесь двустороннюю критическую область, если зафиксирована двусторонняя альтернатива? 
Да, можно, тогда критическая область будет иметь вид:

$$
[0; F(\text{df}_1, \text{df}_2, 1-\frac{\alpha}{2})] \cup [F(\text{df}_1, \text{df}_2, \frac{\alpha}{2}), +\infty)
$$
