### Задача 1

Дано:

* $n = 16$, $m = 15$, $s_1 = 10$, $s_2 = 5$

Решение:

Наблюдаемое значение статистики критерия: $F_{набл} = 100/25 = 4$

Пункт a) 

* Гипотезы: $H_0: \sigma^2_1 = \sigma^2_2$ против $H_1: \sigma^2_1 > \sigma^2_2$ при $\alpha = 0.1$
* $F_{крит} = \text{F}(\text{df}_1 = 15, \text{df}_2 = 14, \alpha = 0.1) = 2.01$, критическая область $[2.01; +\infty)$, наблюдаемое значение лежит в ней, $H_0$ отвергается, первая дисперсия больше второй

Пункт b) 

* Гипотезы: $H_0: \sigma^2_1 = \sigma^2_2$ против $H_1: \sigma^2_1 \ne \sigma^2_2$ при $\alpha = 0.05$
* $F_{крит} = \text{F}(\text{df}_1 = 15, \text{df}_2 = 14, \alpha/2 = 0.025) = 2.94$ и $\text{F}(\text{df}_1 = 15, \text{df}_2 = 14, 1-\alpha/2 = 0.975) = 0.35$, критическая область $(0, 0.35] \cup [2.94; +\infty)$, наблюдаемое значение лежит в ней, $H_0$ отвергается, дисперсии не равны

**Важно.** Чем отличается построение критической области в случае сравнения дисперсий и ANOVA? 

Когда мы сравниваем две дисперсии, мы понимаем, что первая дисперсия может быть больше второй, и наоборот. Мы можем большую дисперсию разделить на меньшую (получить значение выше 1, теоретически до $+\infty$), и меньшую на большую (получить значение ниже 1, теоретически от 0 до 1), отсюда возникает идея учета двух хвостов распределения.

Чтобы далее не привязываться к тому, какая из дисперсий больше, зафиксируем обозначения:

* первая выборка: дисперсия $s_1^2$, объем $n_1$
* вторая выборка: дисперсия $s_2^2$, объем $n_2$

Тогда, если мы определим $F_{набл} = \frac{s_1^2}{s_2^2}$, $\text{df}_1 = n_1 - 1$, $\text{df}_2 = n_2 - 1$, получим:

* при двусторонней альтернативе $H_1: \sigma^2_1 \ne \sigma^2_2$ критическая область $[0, \text{F}(\text{df}_1, \text{df}_2, 1-\alpha/2)] \cup [\text{F}(\text{df}_1, \text{df}_2, \alpha/2); +\infty)$
* при правосторонней альтернативе $H_1: \sigma^2_1 > \sigma^2_2$ критическая область $[\text{F}(\text{df}_1, \text{df}_2, \alpha); +\infty)$, (смотрим на отклонения от 1 вправо, так как правостороннюю альтернативу выбираем, когда первая дисперсия больше второй);
* при левосторонней альтернативе $H_1: \sigma^2_1 < \sigma^2_2$ критическая область $[0; \text{F}(\text{df}_1, \text{df}_2, 1-\alpha)]$ (смотрим на отклонения от 1 влево, так как левостороннюю альтернативу выбираем, когда первая дисперсия меньше второй).

Когда мы сравниваем средние с помощью ANOVA, мы всегда оцениваем, во сколько раз межгрупповая дисперсия превышает внутригрупповую, здесь порядок деления при вычислении $F_{набл}$ изменить нельзя. Отсюда критической областью считается только диапазон значений выше $F_{крит} = F(\text{df}_1, \text{df}_2, \alpha)$, так как для отвержения нулевой гипотезы необходимо, чтобы значение F-статистики существенно превышало 1.

Итого: несмотря на то, что в альтернативной гипотезе в ANOVA фигурирует $\ne$, уровень значимости $\alpha$ мы никогда не делим пополам, потому что критическая область в любом случае односторонняя.

### Задача 2

Дано:

* $n_1 = 6$, $n_2 = 7$, $n_3 = 5$, $n = 18$ (число наблюдений), $k = 3$ (число групп)
* $SS_\text{within 1} = 108$, $SS_\text{withinn 2} = 94$, $SS_\text{within 3} = 40$
* $\bar{y}_1 = 15$, $\bar{y}_2 = 22$, $\bar{y}_3 = 26$
* $\alpha = 0.05$, сравниваем средние в трех группах

Решение:

* Гипотезы: $H_0: a_1  = a_2 = a_3 = a$ против $H_1: \exists j \text{ } a_j \ne a$
* Считаем общее среднее: $\bar{y}=\frac{15 \cdot 6 + 22 \cdot 7 + 26 \cdot 5}{18} = \frac{374}{18} \approx 21$
* Внутригрупповая дисперсия: $\text{SS}_{within} = \text{Residual SS} = \text{RSS} = 108 + 94 + 40 = 242$
  и $\text{Mean RSS} = \text{RSS} / (n - k) = 242/15 = 16.13$
* Межгрупповая дисперсия: $\text{SS}_{between} = \text{Model SS} = \text{MSS} = (15-21)^2 \cdot 6 + (22-21)^2 \cdot + 7 + (26-21)^2 \cdot 5 = 348$,
  и $\text{Mean MSS} = \text{MSS} / (k - 1) = 348/2 = 174$
* Наблюдаемое значение статистики критерия: $F_{набл} = \text{Mean MSS}/\text{Mean RSS} = 174 / 16.13 \approx 10.79$
* Критическое значение статистики критерия: $F_{крит} = \text{F}(\text{df}_1=2, \text{df}_2=15, \alpha = 0.05) = 3.682$,
  критическая область: $[3.682, +\infty)$, $H_0$ отвергается, средние отличаются (как минимум в одной паре групп)
* $R^2 = \frac{MSS}{TSS} = \frac{348}{348 + 242} \approx 0.59$

### Задача 3

Дано:

* $k=6$, $n=150$
* Межгрупповая дисперсия: $\text{MSS} = 1853.5$
* Внутригрупповая дисперсия: $\text{RSS} = 2200.5$
* $\alpha = 0.10$, сравниваем средние в шести группах

Решение:

* Гипотезы: $H_0: a_1  = \dots = a_6 = a$ против $H_1: \exists j \text{ } a_j \ne a$
* $\text{Mean MSS} = 1853.5/5 = 370.7$
* $\text{Mean RSS} = 2200.5/144 = 15.28$
* Наблюдаемое значение статистики критерия: $F_{набл} = \text{Mean MSS}/\text{Mean RSS} = 24.26$
* Критическое значение статистики критерия: $F_{крит} = \text{F}(\text{df}_1=5, \text{df}_2=144, \alpha = 0.1) = 1.889$,
  критическая область: $[1.889, +\infty)$, $H_0$ отвергается, средние отличаются (как минимум в одной паре групп)
* $R^2 = \frac{MSS}{TSS} = \frac{1853.5}{1853.5 + 2200.5} \approx 0.46$
